# 🐾 Hungry Husky

A simple dining dollars budget tracker for Northeastern University students.
_(not officially affiliated with NEU)_

## Features

- Select your meal plan from the dropdown
- Enter how much you've spent
- See your remaining balance broken down by day/week/month
- Visual progress bar shows budget usage

## Development

```bash
npm install
npm run dev
```

## Built With

- React + Vite
- Tailwind CSS
- Claude Opus 4.5

## Meal Plan Data

Based on NEU's 2025-2026 Traditional Meal Plans:
- NU-Unlimited: $400 Dining Dollars
- NU-225: $600 Dining Dollars
- NU-180: $300 Dining Dollars
- NU-150: $200 Dining Dollars
- NU-100: $200 Dining Dollars
